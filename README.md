# University-Empirical-Project

This project is an early work in progress and aims to answer the following research question: How has investor sentiment regarding energy markets changed with oil volatility? Studied since the US 2025 incursion into Iran up to present day.

Premade data will be uploaded as NLP models take a long time to run (will add via OneDrive or Google Drive).

If you would like to run it yourself, please ensure you have sufficient VRAM to accelerate the NLP models. I am running an RTX 4070 Super (desktop).
