# University-Empirical-Project

This project is an early work in progress and aims to answer the following research question: Can oil price movements with regards to the Iran war predict next day financial news sentiment?

Premade data will be uploaded as NLP models take a long time to run (will add via OneDrive or Google Drive). If you would like to run it yourself, please ensure you have sufficient VRAM to accelerate the NLP models. I am running an RTX 4070 Super (desktop).
(Time to run should now be quite feasible on any laptop, although more powerfull ones with accelerate it substancially).
